# MarkDown

## 简介
[markdown中文网](http://markdown.p2hp.com/basic-syntax/)<br/>
> Markdown 是一种轻量级标记语言，创始人是约翰・格鲁伯（John Gruber）。<br />
> 它允许人们 “使用易读易写的纯文本格式编写文档，然后转换成有效的 HTML 文档”。<br />
* 本文记录一些常用的markdown语法用于查阅<br />

## 分割线
* 我们可以通过输入三个或更多的「减号 -」、「星号 *」、「下划线 -」的方式创建一条相当于 HTML 语法中 `<hr/>` 一样的分隔线。
当然也可以直接使用 `<hr/>`。
* 分割线的样式可以通过css来进行控制。
```
//三种分割线的写法
---
***
___
```

## 列表
### 无序列表
无序列表可以通过使用`*` `-` `+`来定义。
```
//使用星号生成无序列表

* 项目1
* 项目2
* 项目3

//使用加号生成无序列表

+ 项目1
+ 项目2
+ 项目3

//使用减号生成无序列表

- 项目1
- 项目2
- 项目3
```
### 有序列表
* 直接手写数字实现<br/>
其他的写法基本不会使用。
```
//使用数字实现有序列表

1. 项目1
2. 项目2
3. 项目3
```

## 强调
使用`*`和`_`可以定义强调的样式<br/>
使用这种定义后，显示文字会变成斜体<br/>
```
// 用法
*前端测试*
_周末快乐_
```

## 字体字号
可以使用 `<font>` 标签的 `face` 属性修改文字字体。<br />
在`style`标签里可以直接写样式控制<br/>
```
<font face="HEI">这里是黑体</font>
<font face="KAI">这里是楷体</font>
<font face="Microsoft YaHei">这里是微软雅黑</font>
<style>p {font-size: 2rem;}</style>
```

## 前景色 背景色
可以使用` <font>` 标签的 `color` 属性修改文字颜色。
```html
<font color="red">红色</font>
<font color="green">绿色</font>
<font color="blue">蓝色</font>
<font color="rgb(200, 100, 100)">使用 rgb 颜色值</font>
<font color="#FF00BB">使用十六进制颜色值</font>
```
效果如下：<br />
<font color="red">红色</font><br />
<font color="green">绿色</font><br />
<font color="blue">蓝色</font><br />
<font color="rgb(200, 100, 100)">使用 rgb 颜色值</font><br />
<font color="#FF00BB">使用十六进制颜色值</font><br />

`<p>hello world<p/>`

## 代码块
1.在行首添加 4 个连续的空格，可将行内容定义为代码块。<br/>
2.在代码块的开始和结束处连续使用3个反引号<br/>
**注意：方法1需要输入4个空格后换行开始写代码**<br/>
```
// 实际写的时候引号要去除
" ``` "
// 你的代码
" ``` "
```

## 表格
Markdown 表格由 「竖线 |」、「减号 -」、「冒号 :」三种符号组成。<br />
* **竖线** 用来定义列，每两个竖线之间为一个单元格元素；<br />
* **减号** 用来定义分割线，也就是分割表头和数据体；<br />
* **冒号** 配合减号使用，用于定义列数据的对齐属性。<br />
```
|学号|姓名|
|----|----|
|1|张三|
|2|李四|
|3|王五|
```
效果如下：<br/>
|学号|姓名|
|----|----|
|1|张三|
|2|李四|
|3|王五|

#### 修改表格的对齐方式
通过「冒号 :」配合「减号 -」使用<br />
```
|左对齐|居中对齐|右对齐|
|:--|:--:|--:|
|1|张三|17岁|
|2|李四|18岁|
|3|王五|19岁|
```
效果如下：<br/>
|左对齐|居中对齐|右对齐|
|:--|:--:|--:|
|1|张三|17岁|
|2|李四|18岁|
|3|王五|19岁|
